• Welcome to the Gamers' Zone!
  • Click the Start button to start the game
  • The goal is to break all of the Rectangles
  • You have only 3 lifes for each level
  • Use the Left and Right arrows to move the plate
  • Press the Pause button to stop the game
  • Press the Resume button to continue the game
  • If you lose, don't worry press the Restart button twice to start over!
